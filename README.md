# Quick Start                                                                                                                                                                                            
### Install as a package

```bash
git clone https://github.com/igudav/SeqNAS.git
pip3 install . 
```

### Use in a docker container

Without installing as a package

```bash
git clone https://github.com/igudav/SeqNAS.git
cd docker
docker-compose build && docker-compose up
```
